【プロジェクト名】
星の作物キャッチゲーム（エルフの森 二次創作ミニゲーム）

【ゲーム全体の流れ】
1. オープニング画面（リトライ／履歴／開始ボタン）
2. フェーズ1：マスターが種をばら撒く → ザルでキャッチ（SeedCatchPhase）
3. フェーズ2：エルフが星の粉／蒼星の粉／毒瓶を落とす → ザルでキャッチ
4. フェーズ3：拾った種を一覧表示 → 指タップで粉を最大10回振り分ける（ページ分割あり）
5. フェーズ4：種がグラグラ → 星の作物 or ハズレに成長（成長アニメ）
6. フェーズ5：結果表示（取得内容 → ELF換算 → 日本円換算）
7. リトライ・履歴表示・シェアの選択

【画面構成】
- スマホ縦型：縦720px / 横360px
- スワイプでザル移動（X軸）／タップで粉を振る／画面全体が操作エリア
- セリフ演出：フェーズ切り替え時にキャラ画像＋セリフをバナー表示（JSON管理）

【BGM・SE】
- BGMはプレイ中のみ1曲、リザルト時は停止
- 効果音：ゲーム開始、粉・種投下／キャッチ音、抽選音、成長音、カウントアップ音など
- ミュートボタンは画面右上

【抽選仕様（drawTable.json）】
- 粉レベル0（未使用）：100%ハズレ
- 粉レベル1：ハズレ50%、以後半減（25%、12.5%、6.25%、…）
- レアリティ順：金 > エメラルド > 銀 > サファイア > 銅 > ガラス > ハズレ（エルフシルバー）
- 蒼星の粉は自動的に+5回分として星の粉に加算される

【ELF換算方法】
- 金属系（銅・銀・金）は「JPY換算金額 ÷ ELF価格（例0.94）」でELF数算出
- 石系（ガラス30ELF、サファイア1,000ELF、エメラルド10,000ELF）は固定
- ELF価格は bitFlyer API などから fetchElfPrice.js で取得

【UIコンポーネント】
- ザルキャッチのザル画像：マスター用（img_master）／エルフ用（img_elf）
- 粉・種・作物アイコン：img_seed、img_powder、img_poison、img_crop_XXX（種類別）
- 結果画面：3段階表示（取得リスト → ELF → 円）にモザイクフェード＋カウント演出

【セーブ機能・履歴】
- プレイ結果はローカルに保存（historyManager.js）
- 履歴はオープニング画面または結果画面から参照可能

【その他】
- 毒瓶：取得時に星の粉 -2 としてカウント（星の粉の1/5出現率）
- 粉かけ演出：種の背後に光るパーティクル、回数でサイズ変化
- 種キャッチ時間：10秒、1.5秒ごとに5個ずつ発射、画面上に残らなくなったら終了
